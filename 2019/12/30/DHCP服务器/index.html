一、修改服务器的IP地址：

网卡配置文件存放位置：/etc/sysconfig/network-scripts/
1、 修改网卡配置文件：ifcfg-ens33
2、 启用网卡: ifup ens33 禁用网卡：ifdown ens33

二、安装DHCP服务

​ yum install dhcpd

三、配置DHCP服务器

DHCP服务的配置文件：/etc/dhcp/dhcpd.conf
如果要配置DHCP需要查看样例件参考，进行配置：
/usr/share/doc/dhcp*/dhcpd.conf.example
也可以通过man dhcpd.conf查看帮助手册进行配置
检查配置语法格式是否有误：dhcpd -t

1、修改dhcpd.conf配置文件：

option domain-name-servers 61.139.2.69 设置DNS服务器选项，

Subnet 192.168.10.0 netmask 255.255.255.0 用于定义作用域

​ Range 192.168.80.21 192.168.80.254 可供分配的IP地址范围

​ Option routers 192.168.80.2 网关

​ default-lease-time 86400 默认租约时间，单位为秒

​ max-lease-time 172800 最大租约时间，一般设置为默认租约时间的2倍

2、重新启动dhcp服务

systemctl restart dhcpd

四、客户端配置

方式一：通过命令提示符
>ipconfig /release
>ipconfig /renew
>ipconfig /all
